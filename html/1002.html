View Component


Untuk bisa mengikuti implementasi bisa buka Snack Expo atau lewat CLI


Yang Akan Dipelajari


Bermain dengan komponen View React-Native


Pembelajaran


View adalah komponen paling dasar di React-Native sederhananya digunakan untuk merender bidang persegi kosong.


Penggunaan


Import dulu `View dari module react-native dengan cara

import React from 'react' import { View } from 'react-native'

dan bisa digunakan di fungsi render seperti berikut, ada bukaan tag

<View>

dan tutupan

</View>

seperti ini

render() { return ( <View> </View> ) }

Styling


View dapat diatur stylenya seperti berikut contoh masukkan didalam return dari render seperti sebelumnya...

<View style = {{ backgroundColor: 'rgb(230,230,230)', flex: 1 }} > </View>


Nesting


View juga bisa dinesting, yang artinya menggambar View di dalam View misal...

<View style = {{ backgroundColor: 'rgb(230,230,230)', flex: 1 }} > <View style = {{ backgroundColor: 'deepskyblue', height: 60 }} > </View> <View style = {{ backgroundColor: 'pink', height: 150 }} > </View> </View>

Lihat working example




Row-Direction


View juga bisa merender secara horizontal / kesamping contohnya dengan atribut flexDirection: 'row' di View parent / View terluar

<View style = {{ backgroundColor: 'rgb(230,230,230)', flex: 1, flexDirection: 'row' }} > <View style = {{ backgroundColor: 'deepskyblue', height: 60, width: 150 }} > </View> <View style = {{ backgroundColor: 'pink', height: 60, width: 60 }} > </View> </View>
