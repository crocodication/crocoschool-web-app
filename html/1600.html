Instalasi via CLI



Untuk penggunaan CLI ini bisa saja digunakan untuk orang yang awam, persyaratannya yaitu install:


1. Visual Studio Code untuk media coding


2. Node.js LTS


3. Android Studio


4. JDK 8


5. "Install Android SDK" dan "Setting JAVA_HOME" jika belum, bisa search di google karena bisa jadi ada perbedaan di masing - masing device


6. Untuk pengguna MacOS saja, Install XCode dan CocoaPods


7. Untuk pengguna MacOS saja, Install Brew dan setelah install jalankan perintah "brew install watchman" di terminal


8. Setelah langkah sebelumnya dilalui, pada Command Prompt / Terminal sekarang anda bisa buat project dengan ketik "npx react-native init NAMA_APP_ANDA"


9. Setelah inisialisasi project selesai, anda bisa buka folder project dari Visual Studio Code dan mulai edit file "App.js"


10. Untuk coba menjalankan aplikasi, aktifkan Opsi Pengembang di device smartphone, aktifkan USB Debugging, hubungkan smartphone dengan USB ke komputer / laptop dan aktifkan USB Media Transfer Protocol.


11. Buka Command Prompt / Terminal MacOS, "cd" (change directory) ke folder project dan jalankan npx react-native run-android


12. Aplikasi akan terinstall di smartphone anda, dan setiap perubahan yang anda lakukan di Visual Studio Code akan langsung nampak perubahannya selama USB terhubung.



Dan itulah langkah - langkah menginstal react-native via CLI.
