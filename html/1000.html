Introduction to React-Native



React-Native adalah framework JavaScript yang digunakan untuk membuat aplikasi cross-platform Android dan iOS.

Dengan kata cross-platform itu berarti kita hanya butuh satu kali coding untuk menghasilkan 2 output di Android dan iOS, tanpa harus coding di masing - masing dua kali.


React-Native adalah framework buatan Facebook, sejak munculnya pada tahun 2015, React-Native merupakan framework yang ready, dewasa dan sudah terbukti digunakan dalam produksi aplikasi seperti Instagram, Facebook, Uber, Discord, Pinterest dll, menggunakan aplikasi React-Native.

Mulai tertarik ingin tahu React-Native? dan ingin tahu juga bisa coding React-Native?

Media coding React-Native


Ada 2 cara untuk coding React-Native, yaitu menggunakan:

1. Snacks Expo

2. CLI (Command Line Interface) seperti Command Prompt(CMD) atau Terminal Mac / Linux + Teks Editor



Via Expo

1. Daftar akun expo

2. Login akun Expo


3. Jika sudah login buka Snack Expo




4. Anda bisa mulai mengedit App.js dan menyimpannya dengan Control + S atau Command + S di MacOS
