Membuat Single Screen di React-Native


Struktur Layar


Di React-Native untuk membuat satu layar, cukup buka "App.js" hapus semua dan ketik struktur kodenya seperti berikut

import React from 'react' import { View } from 'react-native' export default class extends React.Component { render() { return ( <View style = {{ backgroundColor: "cyan", flex: 1 }} > </View> ) } }

Penjelasan Struktur Layar


Dalam pembuatan layar butuh deklarasi class, dari contoh di atas yaitu

export default class extends React.Component { }

export

diperlukan karena compiler React-Native perlu import dari apa yang kita export


default

diperlukan untuk mengetahui class mana yang secara default diimport React-Native


extends React.Component

yang berarti class menggunakan komponen React yang tadi sudah diambil dari import React from 'React


Dan perlu diingat jika ada bukaan { pasti diikuti tutupan }


Class sudah dibuat, lalu untuk merender memakai fungsi render() di dalam class seperti contoh dari di atas yaitu

render() { return ( <View style = {{ backgroundColor: "cyan", flex: 1 }} > </View> ) }
render() { }

merupakan fungsi rendernya, diingat lagi yang mana bukaan ( juga diikuti tutupan ), kemudian


return ( )

untuk didalamnya kita dapat mendefinisikan apa yang diteruskan untuk dirender, lalu

<View style = {{ backgroundColor: "cyan", flex: 1 }} > </View>

ini contoh komponen view yang dirender, deklarasinya ada bukaan tag

<View>

dan diikuti tutupan tag

</View>

di bukaan dapat diisi properti yang tersedia, cara penulisan properti yaitu nama properti nya diikuti "=" lalu isi value nya di dalam buka tutup kurung kurawal contoh:


Properti integer

propertiInteger = {5}

Properti string

propertiString = {"Hello"}

Properti objek JSON

propertiJSON = {{ atribut: "Isi atribut" }}

ini sama yang tadi kita pakai diatas untuk membuat properti style di view

style = {{ backgroundColor: "cyan", flex: 1 }}
backgroundColor: "cyan",

membuat warna background view menjadi warna cyan


flex: 1

menspesifikan ukuran view flex berisi rasio 0 sampai 1, 0 kosong, 0.5 merupakan setengah ruang layar yang tersedia, 1 merupakan keseluruhan layar yang tersedia


itulah secara lengkap sederhananya pembuatan class Screen di React-Native yang mana kode di atas tadi

import React from 'react' import { View } from 'react-native' export default class extends React.Component { render() { return ( <View style = {{ backgroundColor: "cyan", flex: 1 }} > </View> ) } }

digunakan untuk membuat single screen yang merender view penuh satu layar dengan warna background cyan
